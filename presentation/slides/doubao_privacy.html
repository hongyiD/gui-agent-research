豆包手机的隐私保护设计

系统级架构带来的原生安全优势

非像素级读取

利用系统底层权限，直接获取 UI 控件树 (View Tree) 和元数据。Agent 无需"看"到屏幕截图即可理解界面结构，从源头避免了视觉信息的过度采集。

System API Access

端侧隐私脱敏

在必须进行云端推理时，端侧安全模型会先行介入。自动识别并模糊处理身份证号、银行卡、人脸等敏感信息，确保上传数据的脱敏性。

On-Device AI Filter

用户主权管控

提供显性化的状态栏提示和物理级中断机制。用户可以实时看到 Agent 正在读取什么，并随时通过语音或按键终止操作。
