# AutoGLM 性能评测

**发布日期**: 2025年12月16日

**作者**: Manus AI

---

## 1. 概述

为了全面评估 AutoGLM 的能力，其研发团队在多个公开和自建的基准测试集上进行了广泛的实验。这些测试覆盖了 Web 浏览器和 Android 手机两大主流 GUI 场景，旨在衡量智能体在真实世界任务中的**任务成功率**、**泛化能力**和**效率**。本报告将汇总并分析 AutoGLM 在各项评测中的核心性能指标 [1]。

## 2. Web 浏览器场景性能

在 Web 浏览器场景中，AutoGLM 主要在 **VAB-WebArena-Lite** 和一个包含真实世界任务的数据集上进行了评测。

### 2.1. VAB-WebArena-Lite 基准测试

VAB-WebArena-Lite 是一个新兴的、更具挑战性的 Web 智能体评测基准。与之前的基准相比，它包含了更复杂的动态网页和需要长程规划的任务。

| 模型 | 成功率 (第一次尝试) | 成功率 (第二次尝试) |
| :--- | :--- | :--- |
| GPT-4V (SoM) | 38.6% | - |
| CogAgent-18B | 44.1% | - |
| **AutoGLM (GLM-4-9B)** | **55.2%** | **59.1%** |

从上表可以看出，AutoGLM (基于 GLM-4-9B) 在该基准上显著优于包括 GPT-4V 和 CogAgent 在内的其他顶尖模型。尤其值得注意的是，在允许第二次尝试（即在第一次失败后进行反思和重试）的情况下，其成功率进一步提升至 **59.1%**，这充分证明了其**自进化在线学习框架**在错误恢复方面的有效性。

### 2.2. 真实世界 Web 任务

为了测试其在真实商业网站上的表现，研究团队在 **OpenTable** (餐厅预订) 和 **Expedia** (酒店预订) 网站上收集了 100 个任务。评测结果显示，AutoGLM 在这些高度动态和复杂的真实世界网站上表现出色。

-   **OpenTable**: 取得了 **96.2%** 的任务成功率。
-   **Expedia**: 取得了 **82.5%** 的任务成功率。

如此高的成功率表明，AutoGLM 不仅在学术基准上表现优异，更具备了在真实、不受控的商业环境中完成实际任务的强大能力。

## 3. Android 手机场景性能

在 Android 手机场景中，AutoGLM 的评测主要集中在 **AndroidLab** 基准和一系列主流中文应用上。

### 3.1. AndroidLab 基准测试

AndroidLab 是一个用于评估 Android 智能体在多种应用中执行任务能力的标准化基准。

| 模型 | 成功率 |
| :--- | :--- |
| AppAgent | 10.8% |
| GPT-4V (SoM) | 20.3% |
| **AutoGLM (GLM-4-9B)** | **36.2%** |

结果显示，AutoGLM 的性能远超之前的 SOTA 模型 AppAgent，并且显著优于强大的通用模型 GPT-4V。这表明 AutoGLM 针对 GUI 交互的专门优化（如中间接口设计）在移动端同样取得了巨大成功。

### 3.2. 主流中文应用测试

为了验证其在本地化应用生态中的泛化能力，研究团队在 **50 多个主流中文应用**上创建了一个包含 116 个常见任务的测试集。这些任务涵盖了社交、购物、工具等多个类别。

在该测试集上，AutoGLM 取得了高达 **89.7%** 的任务成功率。这一结果极具说服力，证明了 AutoGLM 具备强大的跨应用泛化能力，能够适应不同应用的 UI 设计和交互逻辑，在多样化的真实场景中稳定地完成任务。

## 4. 性能总结

综合来看，AutoGLM 在 Web 和 Android 两大场景的各项评测中均取得了当前领域的顶尖水平 (State-of-the-Art)。

-   **强大的任务完成能力**: 无论是在学术基准还是真实世界应用中，AutoGLM 都展现了极高的任务成功率。
-   **卓越的错误恢复机制**: 在允许重试的测试中，性能有显著提升，证明了其在线学习框架的有效性。
-   **出色的泛化能力**: 在大量未见过的中文应用上依然能保持高成功率，表明其设计理念和技术路径具有很强的通用性。

这些性能指标充分说明，AutoGLM 是当前 GUI Agent 领域最先进的模型之一，为构建真正可用的、通用的自主智能体奠定了坚实的基础。
